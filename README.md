ZONA MOTOR

Zona Motor adalah platform ulasan otomotif modern yang dirancang untuk memberikan informasi mendalam, akurat, dan visual yang memukau bagi para pecinta sepeda motor di Indonesia. Proyek ini dibangun dengan fokus pada estetika premium, keterbacaan tinggi, dan pengalaman pengguna yang responsif.

FITUR UTAMA

- Desain Modern & Profesional: Menggunakan skema warna Light Mode yang bersih dengan aksen Deep Navy untuk kesan eksklusif.
- Sistem Katalog Terpadu: Halaman produk yang tertata rapi untuk memudahkan penelusuran berbagai model motor.
- Spesifikasi Teknis Detail: Setiap ulasan dilengkapi dengan tabel spesifikasi teknis yang minimalis dan mudah dipahami.
- Navigasi Intuitif: Navbar transparan dengan fitur pencarian real-time untuk akses cepat ke konten.
- Fully Responsive: Dioptimalkan untuk berbagai ukuran layar, mulai dari smartphone hingga monitor desktop resolusi tinggi.

TEKNOLOGI YANG DIGUNAKAN

- HTML5: Struktur semantik untuk SEO yang lebih baik.
- CSS3 (Vanilla): Sistem desain kustom dengan variabel CSS untuk konsistensi.
- Bootstrap 5.3: Framework grid untuk tata letak yang responsif.
- JavaScript (Vanilla): Logika interaksi minimalis dan cepat.
- Google Fonts (Poppins): Tipografi premium untuk meningkatkan profesionalitas.

STRUKTUR PROYEK

  asset/          - Kumpulan gambar unit motor dan aset visual
  styles.css      - Pusat desain sistem dan global styling
  main.js         - Logika interaksi (Search, Animation, Scroll)
  Home.html       - Halaman utama (Landing Page)
  Product.html    - Katalog review motor
  tentang kami.html - Informasi profil Zona Motor
  motor[1-3].html - Detail ulasan per unit motor

PANDUAN INSTALASI LOKAL

1. Clone Repository
   git clone https://github.com/username/Zona-Motor.git

2. Buka Melalui Browser
   Buka file Home.html langsung di browser favorit Anda, atau letakkan folder proyek di dalam direktori htdocs (jika menggunakan XAMPP) dan akses melalui http://localhost/Zona-Motor/Home.html.
